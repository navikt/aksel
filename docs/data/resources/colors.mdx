## Teknisk

### Installering

Alle fargene våre er [definert](https://github.com/navikt/nav-frontend-core/blob/master/less/_variabler.less) i
pakken `nav-frontend-core`, og kan installeres slik:

<Bash terminal copy code={`npm install nav-frontend-core --save`} />
<Bash terminal copy code={`yarn add nav-frontend-core`} />

### Bruk med Less

Hvis du bruker Less kan du bruke disse variablene ved å importere en referanse til `_variabler`-fila i cora-pakken i stilarket ditt:

<Bash
  code={`@import (reference) "~nav-frontend-core/less/_variabler";
.minDiv {
    background: @navRod;
}`}
  copy
  language="css"
/>

## Tilgjengelighet

### Generell bruk av farge

Ved bruk av farger gjelder det et generelt krav om at fargene ikke er det eneste virkemiddel for å
formidle informasjon.

<Preview noCode>
  <Undertittel>Eksempler:</Undertittel>
  <ul>
    <li>
      Lenker kan ikke bare skille seg fra vanlig tekst ved å bruke en annen
      farge, men bør også ha understrek eller andre effekter i tillegg
    </li>
    <li>
      En knapp kan ikke bare skifte farge ved hover/klikk, men bør også endre
      form eller posisjon
    </li>
    <li>
      En notifikasjon om noe positivt/negativt kan ikke bare være farget grønn
      eller rød, men bør også ha et støttende ikon med unik form.
    </li>
  </ul>
</Preview>
