# @navikt/ds-icons

Denne pakken leverer [NAVs ikoner](https://design.nav.no/resources/icons) i en pakke med React-komponenter og SVGer.

## Installering

```sh
// yarn
yarn add @navikt/ds-icons

// npm
npm install @navikt/ds-icons
```
