# Feiloppsummering

Feiloppsummering skal brukes til å gi enkel oversikt over alle feil i et skjema.

<TableOfContents />

## Teknisk

<Bash
  terminal
  copy
  code={`npm install nav-frontend-skjema @navikt/fnrvalidator classnames nav-frontend-js-utils lodash.throttle prop-types react nav-frontend-lenker nav-frontend-lenker-style nav-frontend-core nav-frontend-skjema-style nav-frontend-paneler-style nav-frontend-typografi-style nav-frontend-typografi --save`}
/>

<Npm packName="nav-frontend-skjema" imports="Feiloppsummering" />

## Eksempler

Komponenten skal vises nederst i skjema, men over submit-knappen. Se [mønster for skjema-validering](/patterns/form-validation) for eksempel på bruk.
