# Ekspanderbartpanel

Ekspanderbartpanel er et panel som kan ekspandere eller kollapse innhold. Komponenten bruker npm-pakken React-collapse og extender props fra den.

<TableOfContents />

## Teknisk

<Bash
  terminal
  copy
  code={`npm install nav-frontend-ekspanderbartpanel @types/react-collapse classnames nav-frontend-ekspanderbartpanel-style nav-frontend-chevron-style nav-frontend-core nav-frontend-paneler-style nav-frontend-typografi-style nav-frontend-js-utils lodash.throttle prop-types react react-collapse --save`}
/>

## Endre tekststørrelse

For å endre tekst-størrelsen på åpne/lukke-knappen kan man sende inn en annen
[typografi-type](/components/typografi) via `tittel`-propen.

<Preview
  react={`<Ekspanderbartpanel
    tittel={<Normaltekst>Klikk her for å åpne/lukke panelet</Normaltekst>}
  >
    Panelet vil da ekspandere og vise innholdet.
  </Ekspanderbartpanel>`}
>
  <Ekspanderbartpanel

  >
    Panelet vil da ekspandere og vise innholdet.
  </Ekspanderbartpanel>
</Preview>

## Avansert innhold i åpne/lukke-knapp

Komponentens `tittel`-prop støtter rikt HTML-innhold.

<Preview
  react={`<Ekspanderbartpanel
    tittel={
      <div>
        <Undertittel>Klikk her for å åpne/lukke panelet</Undertittel>
        <Etikett type="suksess">En etikett</Etikett>
        <Etikett type="fokus" style={{ marginLeft: "0.5rem" }}>
          En annen etikett
        </Etikett>
      </div>
    }
  >
    Panelet vil da ekspandere og vise innholdet.
  </Ekspanderbartpanel>`}

## Kontroller komponenten selv

      <EkspanderbartpanelBase
        tittel="Kan også klikke her"
