## Normal

Når modalen åpnes vil den legge seg over det andre innholdet på siden. Modalen kan lukkes igjen ved å klikke på
lukk-knappen, klikke utenfor modalruten eller ved å trykke "Esc" på tastaturet. Innholdet i modalruten kan
bestemmes helt fritt.

<Example>
  <ModalExample />
</Example>

```jsx
<Knapp onClick={() => this.openModal()}>Klikk for å åpne modal</Knapp>
<Modal
    isOpen={this.state.modalIsOpen}
    onRequestClose={() => this.closeModal()}
    closeButton={true}
    contentLabel="Min modalrute"
>
    <div style={{padding:'2rem 2.5rem'}}>Innhold her</div>
</Modal>
```
