# Hvordan ta i bruk Stepper

Stepper er en ny utgave av stegindikator.
Du styrer selv state med `activeStep` og `onStepChange`.

## Eksempel

```jsx
<Stepper activeStep={activeStep} onStepChange={(x) => setActiveStep(x)}>
  <Stepper.Step>Start</Stepper.Step>
  <Stepper.Step>Sunt deserunt qui sit sunt culpa nisi</Stepper.Step>
  <Stepper.Step>Nulla nisi pariatur nulla cupidatat elit.</Stepper.Step>
  <Stepper.Step>
    Nulla laborum proident consequat laborum elit et dolore ut sunt.
  </Stepper.Step>
</Stepper>
```

## Override tags

`Stepper.Step` er en `<a>` by default, men dette kan overskrives.
Endre taggen til det du trenger med `as`-proppen. Se eksempelet under.
Merk at Step uansett wrappes i et `<li>`-element for tilgjengelighet.

```jsx
<Stepper activeStep={activeStep} onStepChange={(x) => setActiveStep(x)}>
  <Stepper.Step
    href=""
    onClick={(e) => e.preventDefault()}
  >{`Dette er default, et <a>-element`}</Stepper.Step>
  <Stepper.Step as="button">{`Dette er et <button>-element`}</Stepper.Step>
  <Stepper.Step as="span">{`Dette er et <span>-element`}</Stepper.Step>
</Stepper>
```
