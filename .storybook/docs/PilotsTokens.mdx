# Tokens

## Spacing scale

We have updated our spacing-scale to now be named `space` and for the values to reflect 1-1 the px-value.
To migrate old `--a-spacing-X`-tokens, you can just take the value and 4x it to get the new `--ax-space-X`-token.

```diff
// Old
--a-spacing-4: 1rem;

// New
--ax-space-16: 1rem;
```

## Global tokens

Our global colors are Neutral, Accent, Success, Info, Warning, Danger, Brand-magenta, Brand-beige, Brand-blue, Meta-purple and Meta-lime.

Beside `neutral` that has a `000`-token,
all other color-tokens are named after the following pattern:

```css
--ax-[role]-000 // Neutral only
--ax-[role]-100
--ax-[role]-100A
--ax-[role]-200

```

## Semantic tokens

We have two sets of semantic tokens, one for unique cases and one for all the roles.

```css
// Unique tokens
--ax-bg-default   // Main page background.
--ax-bg-input     // Background for input-fields.
--ax-bg-overlay   // Background for overlays like Modal.
--ax-bg-sunken    // Background for sunken sections to highlight ex. Cards.
--ax-bg-raised    // Background for raised elements, like cards or dialogs.
--ax-text-logo    // Nav-logo color.
--ax-border-focus // Outline-color for focus-states.
```

For the remaning tokens, they are equal between all the global roles. This means that if you know the token for one role, you know it for all the roles.

```css
// Backgrounds

// Soft is used for subtle backgrounds without state.
--ax-bg-[role]-soft
--ax-bg-[role]-softA

// Moderate is used for subtle elements, with the option for states.
// Should be used with a border in most cases to separate from the background.
// Think Secondary-buttons, Alerts
--ax-bg-[role]-moderate
--ax-bg-[role]-moderateA
--ax-bg-[role]-moderate-hover
--ax-bg-[role]-moderate-hoverA
--ax-bg-[role]-moderate-pressed
--ax-bg-[role]-moderate-pressedA

// Strong is used for elements that should stand out, with the option for states.
// Think Primary-buttons, Checkbox, Chips
// Text on strong should use contrast version of the color.
--ax-bg-[role]-strong
--ax-bg-[role]-strong-hover
--ax-bg-[role]-strong-pressed

// Text
--ax-text-[role]
--ax-text-[role]-subtle
--ax-text-[role]-icon
--ax-text-[role]-contrast

// Borders
--ax-border-[role]
--ax-border-[role]-subtle
--ax-border-[role]-subtleA
--ax-border-[role]-strong
```
