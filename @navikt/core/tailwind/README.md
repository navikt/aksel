# Aksel tailwind-preset

Custom tailwind-preset based on `@navikt/ds-tokens`.

## Installation

```bash
  yarn add @navikt/ds-tailwind
  npm install @navikt/ds-tailwind
```

## Usage

```javascript
// tailwind.config.js

module.exports = {
  presets: [require("@navikt/ds-tailwind")],
};
```
