---
ingress: Ekspanderbartpanel er et panel som kan ekspandere eller kollapse innhold. Komponenten bruker npm-pakken <a href="https://www.npmjs.com/package/react-collapse">React-collapse</a> og extender props fra den.
---

import Ekspanderbartpanel, { EkspanderbartpanelBase } from "./../";

import Etikett from "nav-frontend-etiketter";
import { Normaltekst, Undertittel } from "nav-frontend-typografi";

## Endre tekststørrelse

For å endre tekst-størrelsen på åpne/lukke-knappen kan man sende inn en annen
[typografi-type](/components/typografi) via `tittel`-propen.

<Example>
  <Ekspanderbartpanel
    tittel={<Normaltekst>Klikk her for å åpne/lukke panelet</Normaltekst>}
  >
    Panelet vil da ekspandere og vise innholdet.
  </Ekspanderbartpanel>
</Example>

```jsx
<Ekspanderbartpanel
  tittel={<Normaltekst>Klikk her for å åpne/lukke panelet</Normaltekst>}
>
  Panelet vil da ekspandere og vise innholdet.
</Ekspanderbartpanel>
```

## Avansert innhold i åpne/lukke-knapp

Komponentens `tittel`-prop støtter rikt HTML-innhold.

<Example>
  <Ekspanderbartpanel
    tittel={
      <div>
        <Undertittel>Klikk her for å åpne/lukke panelet</Undertittel>
        <Etikett type="suksess">En etikett</Etikett>
        <Etikett type="fokus" style={{ marginLeft: "0.5rem" }}>
          En annen etikett
        </Etikett>
      </div>
    }
  >
    Panelet vil da ekspandere og vise innholdet.
  </Ekspanderbartpanel>
</Example>

```jsx
<Ekspanderbartpanel
  tittel={
    <div>
      <Undertittel>Klikk her for å åpne/lukke panelet</Undertittel>
      <Etikett type="suksess">En etikett</Etikett>
      <Etikett type="fokus" style={{ marginLeft: "0.5rem" }}>
        En annen etikett
      </Etikett>
    </div>
  }
>
  Panelet vil da ekspandere og vise innholdet.
</Ekspanderbartpanel>
```

## Kontroller komponenten selv

Styr komponent fra ekstern state (controlled components).

<Example>
  <EkspanderbartpanelEksempel />
</Example>

```jsx
<Knapp onClick={() => this.open()}>Åpne</Knapp>
<Knapp onClick={() => this.close()}>Lukke</Knapp>
<EkspanderbartpanelBase
    tittel="Kan også klikke her"
    apen={this.state.apen}
    onClick={() => this.toggle()}
>
    Alt innholdet ditt
</EkspanderbartpanelBase>
```
