# Hvordan ta i bruk ConfirmationPanel

```jsx
<ConfirmationPanel
  checked={checked}
  onChange={() => setChecked(!checked)}
  label="Checkbox label text"
>
  Ipsum voluptate pariatur <Link href="#123">demolink</Link> anim officia minim
  ut mollit voluptate exercitation nulla mollit.
</ConfirmationPanel>
```

<Canvas>
  <ConfirmationPanelExample />
</Canvas>

## Størrelser

Kommer i størrelsene "m" og "s".

```jsx
<ConfirmationPanel
  checked={checked}
  onChange={() => setChecked(!checked)}
  label="Checkbox label text"
  size="s"
>
  Ipsum voluptate pariatur <Link href="#123">demolink</Link> anim officia minim
  ut mollit voluptate exercitation nulla mollit.
</ConfirmationPanel>
```

<Canvas>
  <ConfirmationPanelExample size="s" />
</Canvas>

## Errors

```jsx
<ConfirmationPanel
  checked={checked}
  onChange={() => setChecked(!checked)}
  label="Checkbox label text"
  error="Laborum enim esse quis"
>
  Ipsum voluptate pariatur <Link href="#123">demolink</Link> anim officia minim
  ut mollit voluptate exercitation nulla mollit.
</ConfirmationPanel>
```
