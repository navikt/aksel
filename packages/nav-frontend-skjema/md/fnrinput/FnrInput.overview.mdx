<Example>

```jsx
<form onSubmit={(e) => handleSubmit(e)}>
  <FnrInput
    label="Fødselsnummer (11 siffer)"
    bredde="S"
    value={value}
    onChange={(e) => handleChange(e)}
    onValidate={(val) => setValid(val)}
    feil={submit && !valid ? "Ugyldig fødselsnummer" : undefined}
  />
</form>
```
