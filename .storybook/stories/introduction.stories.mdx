# Ta i bruk `@navikt/ds-react` pakken

## Oppsett

```bash
yarn add @navikt/ds-react @navikt/ds-css
```

Siden stilene ikke importeres direkte i komponentene, så må man selv importere ds-css.
Dette gjøres for å la komponetene brukes i Nextjs prosjekter.

```bash
import "@navikt/ds-css"
```

Dette vil også legge css-variabler på :root. For å få dette til å fungere med IE11, kan denne pakken tas i bruk: https://jhildenbiddle.github.io/css-vars-ponyfill/#/

## Testing

For å støtte testing med eks Jest som ikke tillater ESM moduler, så kan denne configen legges til

```bash
"moduleNameMapper": {
  "@navikt/ds-react(.*)": "@navikt/ds-react/cjs$1",
  "@navikt/ds-icons(.*)": "@navikt/ds-icons/cjs$1"
}
```

Dette vil remappe komponentene (og ikonpakken) til å bruke CommonJs versjonen for testing
