## Normal

Knappen skal vanligvis plasseres oppe i høyre hjørne av et vindu. Normal eller blå variant skal brukes på lyse bakgrunner, og kan brukes omhverandre alt etter hva som passer best med designet.

<Example>
  <Lukknapp>Lukk</Lukknapp>
</Example>

```jsx
<Lukknapp>Lukk</Lukknapp>
```

## Blå

Normal eller blå variant skal brukes på lyse bakgrunner, og kan brukes omhverandre alt etter hva som passer best med designet.

<Example>
  <Lukknapp bla={true}>Lukk</Lukknapp>
</Example>

```jsx
<Lukknapp bla={true}>Lukk</Lukknapp>
```
