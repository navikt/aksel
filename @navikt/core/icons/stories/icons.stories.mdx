# Ta i bruk ikonpakken `@navikt/ds-icons`

Alle ikoner kan bli funnet her https://design.nav.no/designsystem/resources/icons

## Oppsett

```bash
yarn add @navikt/ds-icons
```

```jsx
Import { Expand } from `@navikt/ds-icons`;

<Expand
	title="Åpne meny"
/>
```

Note: Det kan være tilfeller der oppsettet ditt ikke har støtte for SVG, og det må da legges til manuelt i eks en webpack config basert på hva som blir brukt.
