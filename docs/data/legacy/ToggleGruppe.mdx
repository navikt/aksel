# ToggleGruppe

ToggleGruppe er en samling av flere ToggleKnapper. ToggleGruppe tar seg også av styling av ToggleKnapper slik at disse står helt inntil hverandre.

<TableOfContents />

## Teknisk

<Bash
  terminal
  copy
  code={`npm install nav-frontend-toggle classnames nav-frontend-js-utils lodash.throttle prop-types react nav-frontend-toggle-style nav-frontend-core nav-frontend-knapper-style nav-frontend-typografi-style --save`}
/>

<Npm packName="nav-frontend-toggle" imports="ToggleGruppe" />

## Normal

I default varianten kan man bare aktivere èn ToggleKnapp om gangen.

<Preview
  react={`<ToggleGruppe
    defaultToggles={[
      { children: "Mandag", pressed: true },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
  />`}
>
  <ToggleGruppe
    defaultToggles={[
      { children: "Mandag", pressed: true },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
  />
</Preview>

## MultiSelect

For å tillate flere aktive ToggleKnapper samtidig kan man bruke `multiSelect` propen.

<Preview
  react={`<ToggleGruppe
    defaultToggles={[
      { children: "Mandag", pressed: true },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
    multiSelect
  />`}
>
  <ToggleGruppe
    defaultToggles={[
      { children: "Mandag", pressed: true },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
    multiSelect
  />
</Preview>

## Minst én

Bruk `minstEn` propen for å håndheve at det alltid må være èn ToggleKnapp som er valgt.

<Preview
  react={`<ToggleGruppe
    defaultToggles={[
      { children: "Mandag" },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
    minstEn
  />`}
>
  <ToggleGruppe
    defaultToggles={[
      { children: "Mandag" },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
    minstEn
  />
</Preview>

## Kompakt

ToggleGruppe støtter også en `kompakt` variant som kan passe bra hvis ToggleKnappene har lite tekstinnhold eller bare ikoner.

<Preview
  react={`<ToggleGruppe
    defaultToggles={[
      { children: "B", pressed: true },
      { children: <em style={{ paddingLeft: 2, paddingRight: 2 }}>I</em> },
      { children: <span style={{ textDecoration: "underline" }}>U</span> },
    ]}
    multiSelect
    kompakt
  />`}
>
  <ToggleGruppe
    defaultToggles={[
      { children: "B", pressed: true },
      { children: <em style={{ paddingLeft: 2, paddingRight: 2 }}>I</em> },
      { children: <span style={{ textDecoration: "underline" }}>U</span> },
    ]}
    multiSelect
    kompakt
  />
</Preview>
