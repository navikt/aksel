# Hvordan ta i bruk card

## MicroCard

Kan brukes som en vanlig lenke direkte:

```
<MicroCard href="/lenke/til/side">Lenketekst</MicroCard>
```

<Canvas>
  <MicroCard href="/lenke/til/side">Lenketekst</MicroCard>
</Canvas>

Hvis man ønsker å ta i bruk nextjs sin `Link`-komponent:

```jsx
<Link href="/lenke/til/side" passHref>
  <MicroCard>Lenketekst</MicroCard>
</Link>
```
