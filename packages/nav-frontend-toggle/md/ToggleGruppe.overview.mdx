## Normal

I default varianten kan man bare aktivere èn ToggleKnapp om gangen.

<Example>
  <ToggleGruppe
    defaultToggles={[
      { children: "Mandag", pressed: true },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
  />
</Example>

```jsx
<ToggleGruppe
  defaultToggles={[
    { children: "Mandag", pressed: true },
    { children: "Tirsdag" },
    { children: "Onsdag" },
  ]}
/>
```

## MultiSelect

For å tillate flere aktive ToggleKnapper samtidig kan man bruke `multiSelect` propen.

<Example>
  <ToggleGruppe
    defaultToggles={[
      { children: "Mandag", pressed: true },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
    multiSelect
  />
</Example>

```jsx
<ToggleGruppe
  defaultToggles={[
    { children: "Mandag", pressed: true },
    { children: "Tirsdag" },
    { children: "Onsdag" },
  ]}
  multiSelect
/>
```

## Minst én

Bruk `minstEn` propen for å håndheve at det alltid må være èn ToggleKnapp som er valgt.

<Example>
  <ToggleGruppe
    defaultToggles={[
      { children: "Mandag" },
      { children: "Tirsdag" },
      { children: "Onsdag" },
    ]}
    minstEn
  />
</Example>

```jsx
<ToggleGruppe
  defaultToggles={[
    { children: "Mandag" },
    { children: "Tirsdag" },
    { children: "Onsdag" },
  ]}
  minstEn
/>
```

## Kompakt

ToggleGruppe støtter også en `kompakt` variant som kan passe bra hvis ToggleKnappene har lite tekstinnhold eller bare ikoner.

<Example>
  <ToggleGruppe
    defaultToggles={[
      { children: "B", pressed: true },
      { children: <em style={{ paddingLeft: 2, paddingRight: 2 }}>I</em> },
      { children: <span style={{ textDecoration: "underline" }}>U</span> },
    ]}
    multiSelect
    kompakt
  />
</Example>

```
<ToggleGruppe
    defaultToggles={[
        { children: 'B', pressed: true },
        { children: (<em style={{ paddingLeft: 2, paddingRight: 2 }}>I</em>) },
        { children: (<span style={{ textDecoration: 'underline' }}>U</span>) }
    ]}
    multiSelect
    kompakt
/>
```
