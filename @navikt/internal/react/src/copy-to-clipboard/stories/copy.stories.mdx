# Hvordan ta i bruk CopyToClipboard

Innholdet i `copyText` vil bli kopiert til clipboard, mens innholdet i `popoverText`
brukes i popover. Hvis komponenten ikke har noen children (tekst i button) vil da en "navds-sr-only" tekst bli
lagt p√• knappen med tekst "Kopier copyText"

```jsx
<CopyToClipboard popoverText={"Kopierte adresse"} copyText={"0660 OSLO"}>
  Kopier adresse
</CopyToClipboard>
```

## Sizing

```jsx
<CopyToClipboard
  popoverText={"Kopierte adressen"}
  copyText={"0660 OSLO"}
  size="small"
>
  Kopier adresse
</CopyToClipboard>
```
