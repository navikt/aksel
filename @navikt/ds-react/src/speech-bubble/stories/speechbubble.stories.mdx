## Positioning

Speechbubble can be positioned "left" and "right" with the `position`-prop

```jsx
<SpeechBubble illustration="OLA" position="left">
  <Bubble>
    Aute minim nisi sunt mollit duis sunt nulla minim non proident.
  </Bubble>
</SpeechBubble>
<SpeechBubble illustration="OLA" position="right">
  <Bubble>
    Aute minim nisi sunt mollit duis sunt nulla minim non proident.
  </Bubble>
</SpeechBubble>
```

## Changing colors

User can change background-color of bubbles and illustration with `illustrationBgColor` and `backgroundColor` props.

```jsx
<SpeechBubble
  illustration="OLA"
  position="left"
  illustrationBgColor="var(--navds-color-lightblue-10)"
  backgroundColor="var(--navds-color-lightblue-20)"
>
  <Bubble>
    Aute minim nisi sunt mollit duis sunt nulla minim non proident.
  </Bubble>
</SpeechBubble>
```

## Illustrations

The `illustration`-prop accepts `React.ReactNode`.
If used with an illustration of format SVG its handled like in `nav-frontend-veilederpanel`

```jsx
<SpeechBubble illustration={<Illustration />} position="left">
  <Bubble>
    Aute minim nisi sunt mollit duis sunt nulla minim non proident.
  </Bubble>
</SpeechBubble>
```
